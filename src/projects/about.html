a simple portfolio

2019

The first programming language I ever learned was HTML and now I know it just as well as English. This entire website was built from the ground up. It started as a idea in my head to a design mock up in Illustrator and now a fully working website. Its crazy to think that anyone can design a whole website from nothing.

I chose to create a website because its modern, scalable and there is lots of freedom. I also wanted to create a website to show that, a website is not just static, but that it can be just as appealing as a video and can even tell a story.
