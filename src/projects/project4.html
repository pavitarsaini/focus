TRON

html | desktop

a windows helper

2018-2019

Tron is one of my many unreleased apps for windows. It is a very simple yet powerful utility to help users clean and fix Windows problems. When realeased it will be free and open source to let anyone tinker with it.

I got the idea when my family members asked wanted me to fix their computer. Since there is no free and clean windows cleaner out there. So I decided to build one on my own. Tron uses many other open source scripts to solve most windows problems and some custom to code to put it all together in a clean package.

Tron is still under development and set to launch before next year.
